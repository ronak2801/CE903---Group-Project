# AI-Based Recommendation for Food Purchase

This repository contains the complete codebase for a food purchase recommendation system developed as part of the CE902 project.

The system provides intelligent recommendations to assist users in making optimized food purchasing decisions.

---

## Project Structure

The repository is divided into three main components:

### frontend
The user interface that demonstrates the capabilities of the recommendation engine.

### backend
The server-side application that enables communication between the frontend and the recommendation engine.

### rs-engine
The core recommendation engine responsible for developing and generating food purchase recommendations.
