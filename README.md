# AI-Based Recommendation for Food Purchase

This repository contains code bases for a full recommendation system for recommending food purchases as part of CE902 project.

---

## Project Structure

The repository divides the three codebases into three folders:

- **frontend**: The frontend that is used for the purpose of demonstrating the capabilities of the recommendation engine.

- **backend**: The backend that is used for allowing the frontend to interact with the recommendation engine.

- **rs-engine**: The code base that is used for developing the recommendation engine to be used in the backend system.

---
